## 测试目标

BETPlayerCap v2.19.3 是针对 **Backrooms: Escape Together** 的 Steam Windows 版本构建并测试的，
使用了来自可正常工作的测试安装环境的 Unreal Engine **5.7.4** 与 UE4SS **3.0.1** 运行时
文件/签名。

## 更新策略

本模组依赖 UE4SS 签名以及游戏的类名/函数名。即使安装程序仍能成功复制文件，
游戏更新也可能导致它失效。

常见的失效症状：

- 游戏能启动，但 `UE4SS.log` 中没有出现任何 `[BETPlayerCap]` 行。
- UE4SS 报告签名扫描失败。
- 在真实关卡中按键绑定无任何作用。
- 玩家上限 UI 不再能调整到 6 以上。
- 传送/聚集/电梯工具记录对象/类解析失败。

如果在某次 Steam 更新后出现上述任何情况，请使用 `uninstall.bat` 并等待更新后的
发行版本。

## 多人/语音说明

- 玩家上限和房主工具只需要主机/房主安装本模组；客户端如果想使用本地“自身无碰撞”开关，也可以选择安装。
- 语音聊天使用的是基础游戏的 EOS RTC 系统。即使不使用本模组，也曾观察到一些
  特定玩家的语音故障。
- 附带的 `Engine.ini` 仅抑制了一个嘈杂的语音合成欠载（underrun）日志类别；它
  不会改变 EOS 语音联网。

## 反作弊说明

本模组不包含任何反作弊绕过手段。请勿在公开匹配中使用。仅限在所有玩家均知情同意的
私人房间中使用。
